# 📅 Wall Calendar Component

## 🚀 Overview
An interactive wall calendar built using React. Inspired by a physical wall calendar with a clean layout, hero image, and integrated notes section.

## ✨ Features
- 📆 Day range selection (start → end)
- 📝 Notes section (add/edit notes)
- 🖼 Wall calendar UI with image
- 📱 Fully responsive (desktop + mobile)

---

## 🛠 Tech Stack
- React.js
- CSS / Tailwind (if used)
- LocalStorage (for notes persistence)
